# Learning Apps

Each phase that needs runnable code lives in this directory. These apps are intentionally separate so we can compare approaches as the project progresses instead of burying every learning step inside the final application.

## Current Apps

| App | Phase | Stack |
| --- | --- | --- |
| [Browser Basics Task Tracker](phase-1-browser-basics/README.md) | Phase 1 | HTML, CSS, and JavaScript modules |
| [React And Vite Prototype](phase-2-react-vite/README.md) | Phase 2 | React, JavaScript, and Vite |

Phase 3 will add a separate strict TypeScript version of the React prototype.
The final Next.js application remains a later phase.
