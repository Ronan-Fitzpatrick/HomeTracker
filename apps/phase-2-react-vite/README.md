# Phase 2 React And Vite Prototype

The runnable companion to
[Phase 2](../../docs/phases/phase-2-react-vite.md).

This app rebuilds the Phase 1 behavior in React before adding a shopping list
and dashboard. Phase 1 remains a behavioral and visual reference; its imperative
DOM implementation is not copied into this app.

## Commands

```sh
pnpm install
pnpm dev
pnpm lint
pnpm build
```

## Starting Structure

- `src/main.jsx`: React application entry point
- `src/App.jsx`: initial application component
- `src/App.css`: application-level styles
- `src/index.css`: shared tokens and global styles

Component and hook folders will be introduced when the first feature gives
them a clear responsibility.
